The principles of analysis and design include: balance, ensure visual harmony by measuring the symmetry, asymmetry and radial arrangement; alignment, promote readability with element placement; proximity, use white space to create distinct sections and allow elements to breathe; repetition, ensure consistency; contrast, differentiate elements to emphasise distinction; accessibility, adhere to inclusivity guidelines such as Web Content Accessibility Guidelines (WCAG); and design thinking, solve problems creatively by understanding and defining the problem, finding solutions, and creating and testing a prototype.

Website architectural requirements include: provide an optimal user experience (UX); ensure technical performance is robust; ensure content structure has a logical flow; promote search engine optimisation (SEO) and discoverability by utilising sitemaps, internal linking and tags.

Website design structure is the order and hierarchy of web pages, connected with a navigational system. The four primary website design structures include: hierarchical model, arranges content in order of importance; sequential model, guides the user through a step-by-step journey to their chosen destination; matrix model, provides freedom for the user to explore using navigation features; and database model, creates a personalised experience with user inputs.

User-interface design requirements include: develop a simple interface; ensure consistency; utilise common UI elements; ensure page layout is considered carefully based on importance; utilise colour and texture to guide user attention; utilise typography to ensure hierarchy and readability; ensure the user is kept informed of actions, changes and errors; and consider defaults, particularly with forms that can contain completed fields with pre-determined answers. User-interface design production processes include: wireframing, creating a design blueprint to determine the flow and placement of information and elements; visual design, determining the colour theme and typography; and prototyping, developing an interactive sample product to demonstrate design functionality.
