Authentication is the process of verifying the identity of someone or something by comparing the provided credentials against valid credentials stored in a database. Web security is the protection of websites, applications and APIs from security threats such as cross-site scripting (XSS), SQL injection, code injection, phishing, ransomware, viruses, worms, spyware, and denial of service (DoS).

Hypertext Transfer Protocol (HTTP) is an application layer protocol that runs on top of the network protocol layers of the Open Systems Interconnection (OSI) model. HTTP is used to transfer data such as text, images, sound and video between a web browser and web application or server through requests and responses.

A session is the series of requests and responses between a web browser and web application or server within a given timeframe. A session is tracked using the session token and cookies. Session management ensures secure interactions between a web browser and web application or server.

Stateless programming occurs when a server does not track the identity or actions of a user on a web page. In contrast, stateful programming occurs when a server tracks and stores the identity and actions of a user on a web page to provide a personalised experience for the next session.
