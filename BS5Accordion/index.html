Welcome to HTML/CSS/JS & Bootstrap 5

This multi-page website utilises the Bootstrap 5 framework to implement a navigational system and display the four requested groups of information.

Assessment Requirements

  1. A single home web page as the entry point into the website.
  2. A single contact web page with links to the CITE and SMTAFE websites.
  3. The user can navigate between all web pages using a suitably labelled navigation system.
  4. Navigation can be vertical, horizontal or tabbed using the appropriate Bootstrap framework.
  5. The navigation and theme must be consistent across all web pages.
  6. The user can select an item and the appropriate answer or definition will be displayed.
  7. The content display must be accordion or collapse.
  8. The website must be compatible with all contemporary web browsers.
  9. The website must be compatible with all major devices.
  10. The website must be WCAG compliant where appropriate.
  11. The development must fully utilise all aspects of the Bootstrap framework version 5.
