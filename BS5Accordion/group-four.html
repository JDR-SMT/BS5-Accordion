The three primary control structures in programming include: sequence, the linear and ordered execution of code; selection, the conditional execution of code, such as if-else and switch-case statements; and iteration, the conditional loop execution of code, such as for, while and do-while loops.

Markup languages utilise symbols and tags to structure, format and define relationships within a document. Hypertext Markup Language (HTML) is an example of a markup language that is used to develop webpages.

Cascading Style Sheets (CCS) is a language that enables the customisation of how properties such as layout, size, positioning, colours and fonts are displayed on HTML elements.

Programming languages are used to develop software programs, each with individualised syntax to adhere to. Syntax is the set of rules that detail how write and structure the programming language.
